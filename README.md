SMARTPHONE USAGE, PRODUCTIVITY & WELL-BEING ANALYSIS
Project Overview
This project analyzes smartphone usage patterns to understand their relationship with work productivity, sleep duration, and stress levels across different demographic groups. The analysis was conducted using Microsoft Excel, Microsoft SQL Server, and Tableau Public.
Objectives
	Analyze smartphone usage patterns across different demographic groups. 
	Identify trends between smartphone usage, productivity, sleep, and stress. 
	Present findings through data visualization and SQL analysis. 
Dataset
Approximately 4,800 records after data cleaning 
Variables include: 
•	User ID 
•	Age 
•	Gender 
•	Occupation 
•	Device Type 
•	Daily Phone Hours 
•	Social Media Hours 
•	Work Productivity Score 
•	Sleep Hours 
•	Stress Level 
•	App Usage Count 
•	Caffeine Intake 
•	Weekend Screen Time Hours 

Tools Used
	Microsoft Excel 
	Microsoft SQL Server 
	Tableau Public 
Methodology
Explored and analyzed the dataset using Excel. 
	Used SQL queries to answer business questions and validate findings. 
	Developed an interactive dashboard in Tableau Public to visualize the results. 
Key Findings
1. Business owners recorded the highest average daily smartphone usage, spending an average of 6.54 hours on their phones each day. This suggests that business owners may rely heavily on smartphones for communication, operations, and business management.
2. Users aged 46–55 achieved the highest average work productivity score, with an average score of 5.52. This may indicate that this age group maintains a balance between smartphone usage and workplace productivity.
3. Male users reported a slightly higher average stress level than female users, with an average stress score of 5.51. This suggests a potential difference in stress experiences that could be explored further.
4. Android was the most commonly used device type, accounting for 50.17% of all users in the dataset, indicating that it is the dominant smartphone platform among the participants.
5. Professionals recorded the highest average sleep duration, averaging 6.52 hours per night. This may suggest that individuals in professional occupations maintain relatively healthier sleep habits compared to other occupations.
